# 隐患随手拍系统

全员安全隐患上报与追踪平台，支持手机/电脑同时访问，数据实时共享。

---

## 快速启动

### 方式一：双击启动（推荐）
直接双击 `启动.bat`，等待命令行窗口显示访问地址后，用浏览器打开即可。

### 方式二：手动启动
```bash
cd hazard-system
npm install   # 仅首次需要
node server.js
```

---

## 访问方式

启动后在浏览器打开：

| 设备 | 地址 |
|------|------|
| 本机电脑 | http://localhost:3000 |
| 手机（同一WiFi） | http://[电脑IP]:3000 |

> 电脑IP查看方式：在命令行输入 `ipconfig`，找到「IPv4 地址」

---

## 功能说明

### 普通用户
- **上报页面**：拍照/上传隐患照片，填写位置和描述，提交后获得积分
- **全部页面**：查看所有同事上报的隐患，每10秒自动刷新
- **排行页面**：查看积分排行榜

### 管理员
- 点击底部「管理」进入管理后台
- 默认密码：`admin123`
- 可操作：派单给责任人 → 提交整改照片 → 验收结案
- 可删除已上报的隐患记录

---

## 积分规则

| 隐患类型 | 积分 |
|---------|------|
| 一般隐患 | 10分 |
| 较大隐患 | 30分 |
| 重大隐患 | 50分 |

## 目录结构

```
hazard-system/
├── server.js       # 后端服务
├── index.html      # 前端页面
├── package.json    # 依赖配置
├── 启动.bat        # Windows一键启动
├── data.json       # 数据文件（自动生成）
└── uploads/        # 上传图片目录
```

---

## 注意事项

1. 手机和电脑必须连接同一个WiFi（同一局域网）
2. 管理员密码在 `server.js` 第9行修改
3. 数据保存在 `data.json`，定期备份该文件
4. 关闭命令行窗口即停止服务
5. 如需外网访问，需配置路由器端口映射或使用内网穿透工具

---

## 修改管理员密码

编辑 `server.js`，找到：
```js
const ADMIN_PASSWORD = 'admin123';
```
将 `admin123` 改为你的新密码，重启服务即可。
